# 첫 스크립트 프로젝트 만들기

이 튜토리얼에서는 VoiceScriptPlayer에서 가장 기본적인 프로젝트를 생성하고 실행하는 방법을 배웁니다.  
완전 초보자도 이 문서를 따라 하면 첫 프로젝트를 만들 수 있습니다.

## 🎯 목표
- 새 프로젝트 생성  
- 프로젝트 폴더 구조 이해  
- 리소스 추가  
- 스크립트 파일 연결  

---

## 1️⃣ 새 프로젝트 만들기

상단 메뉴 → Project → Create New Project  
프로젝트 이름과 저장 경로를 입력하면 아래 기본 구조가 생성됩니다:

/ProjectName  
 ├─ Assets/  
 ├─ Script/  
 └─ Project.vsmeta

---

## 2️⃣ 리소스 추가하기

Assets 폴더에 이미지, 오디오 등 파일을 추가합니다.  
에디터 좌측 Resource Management 탭에 자동으로 반영됩니다.

---

## 3️⃣ 스크립트 파일 만들기

상단 메뉴 → Script → New Script File  
이름: main.script

다음 내용을 입력합니다:

Scene:
  - Text: "프로젝트가 성공적으로 생성되었습니다!"

---

## 4️⃣ 실행 테스트

상단 ▶️ Run 버튼을 누르면 스크립트가 실행됩니다.  
텍스트가 나타나면 성공입니다.
