# 🔗 프로그램 연동 가이드 (Program Integration)

## 1. 개요 (Overview)
VoiceScriptPlayer는 사용자가 개별 API 키를 발급받는 번거로움 없이도, **Patreon 후원 인증**을 통해 통합 서버를 이용할 수 있는 기능을 제공합니다.

후원 인증을 완료하면 **Hailuo(음성 생성)**, **DeepL(번역)** 등의 유료 기능을 할당된 쿼터 내에서 즉시 사용할 수 있습니다.

---

## 2. 패트리온 후원 연동 (Patreon Integration)

계정을 연동하여 후원자 전용 기능을 활성화하는 방법은 다음과 같습니다.

### **단계 1: 연동 시작**
설정 창의 **[후원 연동]** 섹션에서 **"Patreon으로 후원 연결"** 버튼을 클릭합니다. 직접 개인 API 키를 사용하시려면 하단 입력란에 수동으로 입력할 수도 있습니다.

![patreon-connect-button](../images/patreon-step1-connect.png)

### **단계 2: 패트리온 로그인 및 승인**
1. 웹 브라우저가 자동으로 열리면 **"Patreon으로 로그인"** 버튼을 눌러 계정에 로그인합니다.
2. VoiceScriptPlayer가 요청하는 권한(계정 정보 및 후원 상태 확인)을 확인한 후 **"허용"** 버튼을 클릭합니다.

| 1. 로그인 페이지 | 2. Patreon 로그인 | 3. 권한 승인 페이지 |
| :---: | :---: | | :---: |
| ![patreon-login](../images/patreon-step2-login.png) | ![patreon-authorize](../images/patreon-step3-authorize.png) | ![patreon-authorize](../images/patreon-step3-1-authorize.png) |

### **단계 3: 연동 완료 확인**
성공적으로 연동되면 프로그램 내 버튼이 **"후원 연동 끊기"**로 변경됩니다. 이때 하단에 현재 연동된 서비스(Hailuo, DeepL 등)와 잔여 쿼터 정보가 표시되는지 확인하세요.

![patreon-status-success](../images/patreon-step4-success.png)

## 3. 서비스별 서버 활성화 (Server Configuration)

패트리온 연동을 마친 후, 실제로 각 기능을 사용할 때 **연동된 서버를 사용하도록** 설정을 켜주어야 합니다.

### **VoiceScriptServer 사용 설정**
각 AI 서비스(Hailuo, DeepL 등)의 설정 항목에서 **"VoiceScriptServer 사용"** 스위치를 **[켬]** 상태로 변경하세요.

* **스위치 [끔]**: 사용자가 하단에 직접 입력한 개인 API Key를 사용합니다.
* **스위치 [켬]**: 패트리온 연동을 통해 제공되는 통합 서버 API를 사용합니다. (권장)

![setup-api](../images/patreon-useapi.png.png)

## 4. 후원 등급별 혜택 (Subscription Tiers & Benefits)

Patreon 후원을 통해 각 서비스별 AI 크레딧(사용량)을 부여받을 수 있습니다. 부여된 크레딧은 매월 갱신되며, 등급에 따라 제공되는 용량이 다릅니다.

| 등급 (Tier) | 월 구독료 | DeepL (번역) | Hailuo (음성 생성) |
| :--- | :---: | :---: | :---: |
| **🆓 Free Tier** | 무료 | 10,000 chars | 2,000 chars |
| **🌟 Supporter** | $7 / mo | 100,000 chars | 20,000 chars |
| **🧡 Creator** | $20 / mo | 1,000,000 chars | 100,000 chars |
| **🔥 Pro** | $50 / mo | 3,000,000 chars | 300,000 chars |

### 📋 주요 안내 사항
* **자동 중단 시스템**: 설정된 무료/보너스 크레딧 한도에 도달하면 사용이 자동으로 중단되며, **추가 요금이 절대 발생하지 않습니다.**
* **크레딧 갱신**: 모든 크레딧은 매월 구독 갱신 시점에 맞춰 초기화됩니다.
* **정책 변동 안내**: AI 서비스사의 정책에 따라 제공량 및 혜택에 변동이 있을 수 있습니다.

> 🔗 **상세 내용 확인 및 후원하기** > 각 등급별 최신 혜택 및 자세한 사항은 [공식 패트리온 페이지](https://www.patreon.com/posts/introducing-143634103)를 반드시 참고해 주시기 바랍니다.

## ⚠️ 이용 전 주의사항 (Important Notice)

Patreon 멤버십은 **VoiceScriptPlayer의 지속적인 개발과 유지보수를 후원하는 방법**입니다. 제공되는 AI 크레딧은 후원에 대한 감사의 표시로 제공되는 **보너스 혜택**입니다.

* **직접 구독 권장**: AI 서비스(DeepL, Hailuo 등)를 대량으로 혹은 전문적으로 사용하시는 경우, **해당 AI 서비스를 직접 구독하여 API Key를 발급받아 사용하는 것이 비용 대비 훨씬 저렴합니다.**
* **상용 플랜 미대체**: 본 프로그램에서 제공하는 크레딧은 각 AI 서비스의 정식 상용 플랜(Pro 등)을 대체하기 위한 용도가 아닙니다.
* **가성비 안내**: 후원을 통해 제공되는 크레딧은 통합 서버 유지비 등이 포함되어 있어, 서비스사에서 직접 구매하는 것보다 단가가 높게 책정되어 있습니다.

> 💡 **효율적인 사용법**: 가벼운 사용이나 설정의 번거로움을 피하고 싶을 때는 **Patreon 연동**을, 대량의 번역이나 음성 생성이 필요한 작업에는 **개별 API Key** 사용을 권장합니다.

## 5. 문제 해결 (Troubleshooting)

* **❓ 후원 중인데 쿼터가 0으로 나옵니다.**
    * "후원 연동 끊기"를 누른 후 다시 한번 연동을 진행해 보세요. 데이터 동기화에 약간의 시간이 소요될 수 있습니다.
* **❓ "VoiceScriptServer 사용" 옵션이 보이지 않습니다.**
    * VoiceScriptPlayer가 최신 버전인지 확인해 주세요. 구버전에서는 해당 옵션을 지원하지 않을 수 있습니다.
* **❓ 브라우저에서 '허용'을 눌러도 반응이 없습니다.**
    * 방화벽이나 백신 프로그램이 프로그램의 통신을 차단하고 있는지 확인해 주시기 바랍니다.
