# Creator Guide  
シンプルなコンテンツ制作チュートリアル

このガイドは、VoiceScriptPlayer を初めて使用する **クリエイター向け** の入門チュートリアルです。  
以下の手順に従えば、**音声・テキスト・画像・インタラクション** を含む  
小さなプロジェクトを完成させることができます。

---

## 1️⃣ 新しいプロジェクトを作成する

- 上部メニューの **“Create New Project”** を選択します。  
- プロジェクト名と保存場所を入力すると、基本構造が自動生成されます。  
- 作成されたフォルダには `Assets`（リソース）と `Script`（イベント/ロジック）が含まれます。

➡️ 参照: [Create New Project](project/create.md)

---

## 2️⃣ リソースを追加する

- `Assets` フォルダに **音声・画像・動画・Live2D・Spine リソース**を追加します。  
- 左側の **Resource Management** タブでファイルを一括管理できます。  
- 必要に応じて **変数** や **翻訳辞書** もここで登録できます。

➡️ 参照: [Resource Management](editor/resources.md)

---

## 3️⃣ タイムラインを構築する

- タイムラインに **サウンド・字幕・画像・動画** を配置します。  
- ドラッグで位置を調整し、ズームやプレビュー再生が可能です。  
- コピー/貼り付け、スナップ配置など基本編集ツールも利用できます。

➡️ 参照: [Timeline](editor/timeline.md)

---

## 4️⃣ インタラクションとイベントを追加する

- **UI タブ**からボタン・画像・テキストなどの UI 要素を配置します。  
- 各 UI 要素にクリック、比較、変数変更などの **イベント** を設定できます。  
- 簡単なスクリプトで「次のシーンへ移動」「変数を増加」などの挙動を実現できます。

➡️ 参照:  
- [UI](editor/ui.md)  
- [Script](editor/script.md)

---

## 5️⃣ AI 機能を活用する（任意）

- Whisper を使って **自動字幕生成 (STT)**  
- DeepL・LibreTranslate による **翻訳**  
- hailuo・COEIROINK・ElevenLabs による **TTS 音声生成**  
- 翻訳が不自然な場合は **翻訳辞書** で個別に調整できます。

➡️ 参照: [AI Features](ai/whisper.md)

---

## 6️⃣ テスト & 再生

- 上部の ▶️ アイコンを押して **再生モード (Play Mode)** に入ります。  
- 通常 / 手のみ / デバイス推奨 / デバイスのみ のモードを選択できます。  
- マイクが必要な場合は権限確認が表示されます。  
- エラーは画面に表示され、詳細ログは  
  `Documents/VoiceScriptPlayer/Log` に保存されます。

➡️ 参照:  
- [Run](playback/run.md)  
- [Debug & Log](playback/debug.md)

---

## 7️⃣ ビルドとエクスポート

- プロジェクトが完成したら `.vsplayer` 形式でエクスポートできます。  
- このパッケージは **再生専用版 VoiceScriptPlayer** でそのまま再生可能です。

➡️ 参照: [Project Management](project/manage.md)

---

## 🎯 まとめ

ここまで完了できれば、  
すでに **基本的なコンテンツ制作とインタラクション構築** ができるレベルになっています。

VoiceScriptPlayer の強みは、  
**テキスト・音声・画像・UI・インタラクション・デバイス制御を  
1つのプロジェクトに統合できること** です。

次のステップとして、以下を試してみてください：

- ストローク編集  
- デバイス制御  
- 高度なスクリプト  
- UI ↔ Script の連携  
- Live2D / Spine アニメーション制御

自分だけの制作ワークフローを作りましょう！
