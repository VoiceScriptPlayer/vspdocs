    编辑器概览¶

    VoiceScriptPlayer 编辑器（Editor） 是用于构建项目和编辑资源的核心工具。
    本页面介绍编辑器的整体界面结构及各个标签页的作用。

    1. 启动编辑器¶

    创建或打开项目后，编辑器会自动启动。
    编辑器由多个编辑标签页组成，
    每个标签页用于管理不同类型的资源（如音频、视频、Live2D 等）。

    💡 提示：
    VoiceScriptPlayer 同时只能打开一个项目。
    若要打开新项目，请先关闭当前编辑器并返回主界面。

    2. 界面结构¶

    VoiceScriptPlayer 编辑器主要由以下部分组成：
    左侧资源列表、中央播放器（预览区）、底部时间轨与播放控制区。

    区域 说明
    顶部菜单栏 包含 文件、设置 等基本菜单，用于项目保存、发布和环境设置。
    左侧面板（资源列表） 显示当前标签的资源（如音频、视频、Live2D 等）。
    可新增或删除资源。
    中央区域（播放器） 显示所选的音频、视频或 Live2D 模型的实时 预览画面。
    可调整大小或全屏查看。
    底部区域（时间轨 / 时间线） 显示音频或事件的时间轴，可 设置播放区间和起始轨道。
    包括播放、暂停、循环等控制按钮。
    项目信息栏 顶部中央显示当前项目标题及导入的 DLsite 作品信息。
    状态栏（右上角） 显示连接的设备状态（如 TCode、Intiface），可查看“无连接设备”等信息。

    🎮 主要功能区域说明¶

    • 播放器区域
      所有视频、Live2D、UI 元素都会显示在此区域，
      可像实际游戏一样进行 同步预览。
      视频播放、Live2D 动画、UI 交互都会实时反映。

    • 时间轨设置
      在底部时间线可添加轨道或设定起始点。
      点击 设置起始轨道 按钮后，该轨道会作为项目的起始播放点。

    • 资源列表
      左侧显示与当前标签对应的所有资源。

    💡 提示：
    音频与视频标签页的结构完全相同。
    熟悉“资源列表 → 预览 → 时间轨设置”的操作流程后，
    其他标签页的编辑方式也一致。

    3. 编辑标签结构¶

    VoiceScriptPlayer 编辑器包含以下标签页：

    标签名 说明
    🎵 音频 导入音频文件（.mp3, .wav），可进行 字幕生成与编辑、翻译、语音合成 (TTS) 等操作。
    🎬 视频 添加视频文件（.mp4, .mov），并可生成字幕、翻译、编辑同步。
    🧍 Live2D 导入角色模型（.model3.json）及动作文件并进行布局与设置。
    🧩 UI 编辑按钮、图像、文本等 用户界面组件 的布局与属性。
    💬 脚本 使用 变量、触发器、条件语句 定义并控制交互逻辑。
    ✏️ Stroke 编辑基于时间轴的 物理运动轨迹模式。
    ✨ 特殊事件 提供 预定义的特殊事件 模块。
    📦 资源管理 统一管理项目中的 全局资源（变量、图像、Live2D、翻译词典等）。
    🧰 运行时助手 在测试项目时显示调试与运行时信息。

    4. 基本编辑流程¶

    VoiceScriptPlayer 的编辑流程以 音频轨道为核心，同步所有元素。
    按以下顺序进行操作可更自然地完成项目：

    1. 🎵 在音频标签中构建轨道
      导入语音或背景音乐文件（.wav、.mp3）。
      使用 字幕生成（STT） 功能将语音转为文本。
      可拆分区段或剪去无用片段以整理时间轴。
    2. 🎬 在视频或 Live2D 标签中同步场景
      导入视频或 Live2D 模型，并 与音频时间对齐。
      可根据语音区段调整 Live2D 的动作、表情和动画。
      视频可添加字幕或自动翻译已有字幕。
    3. 🧩 在 UI 标签中设计界面
      布置按钮、文本、图像等交互元素。
      可为 UI 元素绑定 脚本事件，如：
      “点击按钮跳转到下一轨道” 或 “显示选项框”。
    4. 💬 在脚本标签中定义逻辑
      使用变量、触发器、条件语句构建 项目的控制流程。
      示例：在特定区段结束后播放 Live2D 动画、切换语音等。
      可统一管理 UI、Stroke、音频之间的交互。
    5. ✏️ 在 Stroke / 特殊事件 标签中同步物理效果
      使用 Stroke 编辑器设置 运动模式与节奏。
      特殊事件标签用于构建高潮或特殊演出效果。
      共享与音频相同的时间轴，可精确调整同步。
    6. ▶️ 预览与调试
      点击底部播放按钮 ▶ 预览整个序列。
      在播放过程中可通过 运行时助手 查看事件与变量状态。

    💡 提示：
    仅修改时间线时需要保存。
    资源的添加或更改会即时生效，无需额外保存操作。

    5. 常用快捷键¶

    快捷键 功能
    Ctrl + S 保存当前项目

    6. 编辑器设置与初始化¶

    • 程序设置（Settings）
      在顶部菜单中选择 设置 → 程序设置，
      可调整 UI 主题、语言、麦克风、AI 参数、快捷键等。

    7. 下一步操作¶

    VoiceScriptPlayer 的所有编辑都以 时间线（轨道） 为核心。
    音频、视频、Live2D、UI、脚本、特殊事件等
    都以时间线为基准进行同步。

    接下来，请先了解 时间线与轨道结构，
    再学习如何在其上放置和编辑资源。

    🎵 各类资源编辑指南¶

    理解时间线后，可继续在各资源标签中进行详细编辑：

    • 音频编辑 — 语音文件、字幕生成 (STT)、轨道配置
    • 视频编辑 — 视频播放、字幕同步、翻译
    • Live2D 编辑 — 角色模型、动作、表情控制
    • UI 设计 — 按钮、文本、界面布局
    • 脚本编辑 — 触发器与变量逻辑
    • 特殊事件配置 — 高潮演出、效果与时序控制
